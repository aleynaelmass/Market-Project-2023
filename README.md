# Laravel Market Project

## About

This is a simple market project built with Laravel 10

## Installation

1. Clone the repository
2. Run `composer install`
3. Run `npm install`
4. Run `npm run dev`
5. Create a copy of your .env file

    `cp .env.example .env`
